# GeoTIFFLayer
Display GeoTIFFs on your Leaflet Map

# Usage
```
let GeoTIFFLayer = require("geotiff-layer-for-leaflet");

let options = {
    geotiff: geotiff, // geotiff object create by geotiff
    opacity: 0.7
};

new GeoTIFFLayer(options).addTo(map);
```
